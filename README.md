# Java Object-Oriented Programming (POO) Exercises

## Overview
This repository contains a structured collection of practical programming exercises and laboratory assignments developed during my coursework at the **Universidade de Aveiro**. 

The main focus of these modules was to understand and implement the foundational pillars of **Object-Oriented Programming (POO)** using Java.

##  Repository Structure
The repository follows a professional IDE project layout, separating source code from compiled binaries:

*   **`/src` (Source Code):** Contains the original `.java` source files, organized chronologically into subfolders for each laboratory session (`aula01`, `aula02`, etc.).
*   **`/bin` (Binaries):** Contains the compiled `.class` bytecode files generated by the Java compiler, maintaining the same lecture-by-lecture hierarchy.

*Note: This repository serves as a personal development sandbox tracking my historical learning progress, including active work-in-progress implementations.*

##  Core OOP Principles Applied
* **Encapsulation:** Designing robust classes with proper data hiding.
* **Inheritance & Polymorphism:** Creating scalable class hierarchies and method overriding.
* **Abstraction:** Implementing abstract classes and interfaces for modular architecture.
* **Exception Handling:** Managing runtime errors to build resilient applications.
